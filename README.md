# 짤 콜렉터

웹페이지의 이미지를 조건에 맞춰 자동 수집하는 Chrome 확장 프로그램입니다.

## 기능

- 확장을 켜면 이후 방문하는 일반 웹사이트의 이미지를 자동 수집합니다.
- 실제 저장은 현재 포커스된 활성 웹 탭에서만 진행합니다.
- 현재 화면에 렌더링되어 사용자가 볼 수 있는 이미지와 배경 이미지만 수집합니다.
- 코드에만 연결되어 있거나, 숨김 처리되었거나, 현재 화면 밖에 있는 이미지는 수집하지 않습니다.
- 기존 `img`, 실제 선택된 `srcset` 현재 소스, 보이는 CSS `background-image` 후보를 찾습니다.
- 최소 너비/높이로 저장 대상을 필터링합니다.
- 이미지 최소 크기는 항상 `480 x 480` 이상입니다.
- 페이지 로드 후 0.5~1초 사이의 랜덤 대기 뒤 수집을 시작하고, 이미지는 하나씩 저장합니다.
- 이미지 저장이 끝난 뒤 다음 이미지까지 0.1~1.5초 사이의 랜덤 대기를 둡니다.
- 같은 호스트에서 다운로드 실패가 3회 연속 발생하면 해당 사이트 수집만 10분 동안 중단합니다.
- 팝업 통계는 오늘 날짜 기준으로 누적하며, 페이지 이동으로 초기화되지 않습니다.
- 오늘 받은 이미지 목록을 원본 파일명, 페이지 전체 주소, 이미지 전체 주소로 저장하고, 같은 원본 파일명은 당일 다시 다운로드하지 않습니다.
- 저장 폴더는 Chrome 기본 다운로드 폴더 아래의 하위 폴더로 지정합니다.
- 수집 중 Chrome 다운로드 UI를 숨길 수 있습니다.

## 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위의 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. 이 폴더를 선택합니다.

## 사용

1. 툴바의 `짤 콜렉터` 아이콘을 누릅니다.
2. 스위치를 켜면 일반 웹사이트에서 자동 수집이 시작됩니다.
3. 수집할 웹페이지를 열거나 이동합니다.
4. `옵션`에서 이미지 크기, 저장 폴더, 수집 대상을 설정합니다.

## 기본값

- 메인 수집 상태: OFF
- 수집 탭: 현재 포커스된 활성 웹 탭만
- 최소 이미지 크기: `480 x 480`
- 반응형 이미지 현재 소스 확인: ON
- CSS 배경 이미지 포함: ON
- 저장 폴더: `Downloads/jjal-collector`
- 다운로드 UI 숨김: ON

## 참고

Chrome 확장 프로그램은 보안 정책상 임의의 전체 경로에 직접 저장할 수 없습니다. 이 확장은 Chrome의 기본 다운로드 폴더 아래에 지정한 하위 폴더를 만들고 이미지를 저장합니다.
